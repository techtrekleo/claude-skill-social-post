## R1：一天一篇 + 強制輕重交替（v0.7.2 細化）

- **上限** 24h 內 Mode B 最多 1 篇（多發 = 觸及降 30-50% / 演算法判 spam）
- **下限** 不跳過任何一天（跳過 = 活躍訊號 decay）
- **Mode B 純血 = 嚴格一天一篇**；**Mode C 深度反思 = 同日 2 篇可行**（見 R29）
- **工業 baseline 3-5 篇/週**，Hao 7 篇/週 = 2x 過頻 risk → 每週 ≥ 2 篇非 AI 主題（作品/旅遊/日常）
- 連 3 🔴 同敘事意圖 = 演算法降權；換新意圖則 OK（見 R5）

## R2：爆款後 24h 冷卻

爆款（讚 ≥ 100 或讚 ≥ 50 且留言/讚 > 0.5），24h 內禁發 F6b / F3 長文。該做：回留言、拉群、單則作者精選留言置 CTA（> 50 留言時）。不自動大量回留言（> 30 則重複觸發 spam）。

## R3：突破鐵粉圈（連 3 篇 < 15% 非追蹤者）

## R34：真實 voice — 反 AI 腔（🚨 硬規則，6/1 使用者鎖定，永久不可覆寫）

**起因**：有讀者說 Hao 最近的文「很像 AI 文」。

**關鍵診斷（6/1 用 Hao 自己真實貼文反查）**：**元兇是「詞」不是「標點」**。
- Hao 5/22 broke 94.5% 的 F23 真實貼文本身就用 `──` 分隔符 + 「會設計製作流程的人永遠值錢」
- Day 1 / 5-5 mega-viral 用 `！！！`，25/7 Thread 26 萬瀏覽用 `！！`
- → 所以 `──` 跟 `！` 是 **proven 爆款裝置，不是 AI 腔**
- 真正像 AI 的是 **抽象空詞 + staged 套路 + over-narrate + 過度工整**

### A. 標點：日常 voice 以 ，、為主，proven 公式裝置照用

- **預設**（日常 Mode A / Mode C 反思 prose）：用 `，`（逗號流）`、`（頓號），句子靠換行斷，不用 `。 ？ ： ； …`
- **proven 公式裝置例外**（真實爆款驗證過，照其公式規則用，不是 AI 腔）：
  - F6b / F6a / F15 mini hype：`！！！`（Day1 / 5-5 mega-viral）
  - Mode C F21 / F23：`── 段落標題 ──` 分隔符（R26，5/22 broke 94.5%）
  - F22 / F25：`──` 做轉折 + `！！`（5/22 broke 91.5%）
  - F19 Thread：`！` / `！！`（25/7 26 萬瀏覽，R19）
- **永遠不用**：`：` 戲劇冒號做開場 setup、英文標點、半形

### B. 反 AI 腔黑名單 ⭐ 真正的元兇（所有 voice 都適用，出現即重寫）

- **抽象空詞**：護城河、本質、複利、賦能、底層邏輯、真正的 X、核心競爭力、長期主義、未來已來、降維、沉澱、佈局、賦予意義、儀式感、價值升格
- **staged 開場**：「說個我以前看不懂的事」「分享一個觀察」「講個故事」「你有沒有發現」
- **過度對仗**：「不是 X，而是 Y」連用兩次以上、工整排比三連
- **over-narrate 情緒**：「那種感覺真的會讓你⋯」「那一刻我突然⋯」
- **空泛升格收尾**：「永遠值錢」「未來已來」式格言 ── 除非像 Hao 真實的「**會設計製作流程的人**永遠值錢」那樣**綁具體技能**，否則砍

### C. 真實感從哪來（正向）

- **具體 > 抽象**：講真的事、真數字、真動作（弄、搞、debug、ship、改、跑）
- **口語詞**：弄、搞、幹嘛、想通、好像、有夠、扯、草、爽
- **自嘲 > 宣言**
- **不對齊 = 人味**：真人深夜不會寫對仗句，會講一半跳下一句

### 範例對照（6/1 深夜訓練 skill 文，real before/after）

- 「建立 30 篇貼文」(4.3 篇/天) = 絕不追，會炸 R1（1/天）+ R5（敘事意圖冷卻）+ voice
- 實證：6/1 本週 8 篇就破 25 萬瀏覽，量沒意義；質暴打量是 Hao 核心優勢

**③ 連 3 指標↑ = 廣推窗口（1-2 週）**
- 分析面板「瀏覽 / 互動 / 淨追蹤」同步↑ → 演算法判定上升星，加碼推廣
- 該窗口優先廣推公式（F23 行業反主流 / F25a manifest / F6b 純血），**避免鐵粉深化篇浪費 momentum**
- 任務目標常設成「複製上週 +4%」→ 維持原打法自然達標，把心力投到唯一新槓桿（Reels）

**6/1 實證**：過去 7 天 250,021 瀏覽（↑37%）/ 3,894 互動（↑114%）/ +169 追蹤（↑74%）；任務面板掛 15 Reels（0/15）+ 30 篇貼文（0/30）+ 260K 瀏覽（複製上週 +4%）。

**狀態**：候選。待 6/2-6/8 實際排 1-2 支 F18 Reel 量測同日觸及增幅，驗證 +50% 後升正式規則。

## R32：集體 framing trigger（v0.9.3，5/26-5/29 三實證）

**任何 Mode B / Mode C 篇用「我們 / 大家一起 / 不只是我」反個人主義 framing = broke 鐵粉圈廣度 +30-50%**。

實證：
- F25a 5/26 ship 系列承諾：**39.7% 非追蹤 broke**（12 儲存）
- F25b 5/28 Line 5000「不是我一個人的功勞」：留言 quality 高
- F25c 5/29「一個人快，一群人遠」階段 review：20.4% 非追蹤

vs 個人 hype：F6b「我殺瘋了」mega-viral 但稀有；F25 集體 framing **普遍可達 broke 鐵粉圈**。

**何時用**：
- 想 broke 鐵粉圈但缺 mega hook 時
- 鐵粉黏著疲勞 → 用「我們」感重燃歸屬感
- ship 系列 / 達 milestone / 階段 review 等敘事時刻

**避免**：
- 不要假大方（要真實 + 具體歸功對象）
- 不要連發 F25 系列（洗鐵粉）
- 不要在 R31 Brand 邊界澄清後立刻用（避免「我們很好我們很棒」連續式自誇）

## R31：Brand 邊界澄清時機（v0.9.2，5/27 F24 實證）

當以下任一觸發 → 用 F24 邊界澄清型：

1. 私訊量爆（回不完）
2. 陌生交友邀請太多
3. 有人誤解你在賣課 / 接案 / 帶貨
4. 鐵粉開始有「跟我們有距離」感
5. 月內 hype + ship 內容過多後需要 reset 信任

**頻率**：每月 1 次最多（太頻繁 = 自我中心感）。

**voice 鐵則**（跟其他 Mode C 不同）：
- 無「！」hype
- 無 TM 級用詞
- 無攻擊性動詞
- 用「── 真的沒有」溫和強調（「──」破折號緩衝）
- 每點 numbered（一、二、三、四、五）
- 謙卑收尾「感謝配合 + 謝謝支持」

**實證 5/27 F24**：2,754 觀眾 / 42.4% 非追蹤 / 173 讚 / **84 留言** / $0.19 收益。鐵粉強認同（「太猛了大神」「感恩分享」）。

## 🎯 Viral 4 條件公式（核心，5 案例 AND 實證）

```
viral = 4 段 4 句結構 + 純血 voice + 全新敘事意圖（R5）+ 黃金時段（R4）
```

4 個 AND 不是 OR，任一缺 = 死。但 **Readability 是隱藏第 5 條件**（5/12 V3 四條全綠仍 fail，因 meta 4 層看不懂）。

## 📝 Readability 三項硬檢查

1. **5 秒非追蹤者讀懂測試**（meta 層數 ≤ 1 最佳，≤ 2 上限）
2. **meta 層數 ≤ 2**（第 3 層以上鐵粉懂新觀眾爬不過）
3. **數字密度 ≤ 3 個/段**（超過 = dashboard report）

## 📦 FB compose 段落硬規則

4 段 4 句貼 FB 必須手動兩次 Enter 保段落（FB 預設壓掉單 Enter）。給草稿時必標註「段間按兩次 Enter」或用 --- 替代。
